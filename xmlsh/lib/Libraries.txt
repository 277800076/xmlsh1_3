SVN and the source bundle do not include the libraries necessary to run xmlsh.
The binary bundle does include these libraries as a convenience for the user.

The following libraries are required to run xmlsh

Log4J:
from: http://logging.apache.org/log4j/1.2/index.html

log4j-1.2.7.jar

Saxon B: version 9.0.0.5
from: http://www.saxonica.com

saxon9.jar
saxon9-s9api.jar
saxon9-dom.jar


Stax Utils from
https://stax-utils.dev.java.net/
Tested with Stabil release
StAX utils 20070216

stax-utils.jar


OPTIONAL FiLES

jline
from 
http://jline.sourceforge.net/


Xerces 2.1 from
http://xerces.apache.org/xerces2-j/install.html
The only file needed is
xercesImpl.jar


Woodstox StAX library from 
http://woodstox.codehaus.org/Download
Apache License version.

woodstox-core-asl-4.0.3.jar
Stax2-api-3.0.1.jar
